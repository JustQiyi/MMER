## 环境要求
- Windows 系统
- Dotnet 8.0 运行环境
- 一个放了模组的文件夹
- 一个脑子

## 如何使用
首先打开程序 (awa
1. (可选) 输入setTargetPath以设定目标路径
2. 输入start，然后输入你需要分离出的模组的文件夹路径
3. 此时会自动识别模组的加载器，将服务端和通用模组复制至目标路径
4. 如果一些模组没有设定环境，会询问是否要复制
5. 完成

## 支持状况
| 加载器 | 符号 | 支持状况 | 理由 |
| ------ | --- | ------- | ---- |
| Fabric | ✔ | 完全支持 | N/A |
| Quilt  | ✔ | 完全支持 | N/A |
| NeoForge | ⚠ | 部分支持 | 目前无法识别部分模组的环境 |
| Forge | ❓ | 未经测试 | 相关代码已完工但未经完整测试 |
| 多加载器 | ❓ | 未经测试 | 尚未测试相关项目 |
